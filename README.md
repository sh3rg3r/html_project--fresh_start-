# “С чистого листа” — HTML/CSS Layout Project

## Project Overview

This project is a static HTML/CSS layout built as a practice of semantic web development and modern styling techniques.

The goal of the project is to learn how to structure a multi-section webpage, organize styles, work with typography, and apply reusable CSS components.

---

## Features

- Semantic HTML structure (header, main, section, footer)
- Modular CSS architecture with separate files for styles and fonts
- Custom font integration (EB Garamond, Inter)
- CSS reset for consistent styling across browsers
- Multi-section layout with different themes (dark and light)
- Use of BEM methodology for class naming
- Background images and text alignment techniques
- Responsive-friendly layout structure
